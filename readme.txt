Filter WooCommerce products by store locations with a location selector for customers.

== Description ==

Filter WooCommerce products by store locations with a location selector for customers. This plugin allows users to view products available in their selected store location, enhancing the shopping experience.

== Installation ==

1. Upload the `multi-location-product-and-inventory-managements` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin settings under WooCommerce -> Store Locations Settings.

== Features ==

- Register custom taxonomy for store locations.
- Filter products based on selected store location.
- Add a location selector modal for users.
- Customize product titles to include the store location.
- Compatible with WooCommerce shortcodes and widgets.
- AJAX support for clearing and checking cart contents.

== Frequently Asked Questions ==

= How do I set up store locations? =

After activating the plugin, navigate to the "Store Locations" section in the WooCommerce settings to add and manage your locations.

= Can I customize the display of product titles? =

Yes! You can configure how the store location appears in product titles under the plugin settings.

= Is there a way to clear the cart when changing locations? =

Yes, the plugin includes an option to clear the cart automatically when a user selects a different store location.
